How to Login/Register

1

Launch SuperLap

When you launch SuperLap, you are greeted with the start page:

SuperLap Start Page
2

Access Login

If you wish to login, click on the login page. You can login with your existing username and password, or if you don't have an account, you can click the register button:

Login Page Interface
3

Register New Account

On the register page, enter your details as prompted and click register to complete registration:

Registration Form
Tip: After registration, you can go to the login page and login using the credentials you just set up.

Getting an Optimal Racing Line

1

Upload Track Image

On the dashboard tab you will see an upload image button. Click on it:

Dashboard with Upload Button
2

Select Track File

Click the upload image button and select a track image from your disk in the file menu that pops up:

File Upload Interface
3

Draw Centerline

After uploading, click on "trace centerline" to draw a rough centerline on the track. This ensures the image processor detects the track correctly:

Drawing the Track Centerline
4

Process the Track

When you're happy with the centerline, click on the "process image" button:

Completed Centerline Ready for Processing
5

View Results

After processing finishes, you'll see the optimal racing line displayed on your track:

Generated Optimal Racing Line

Available Options:

  • Cursor Size: Adjust the size of the racing cursor
  • Animation Speed: Control how fast the cursor moves
  • Trail Length: Modify the length of the cursor trail
  • Animation Toggle: Turn animation on/off to see the full raceline
Animation Controls

Recording Laps in MotoGP

1

Launch Games

Launch MotoGP18, navigate to any race you wish to do and enter the race. Then launch SRT.

2

Select MotoGP18 in SRT

Click on the MotoGP18 tile (indicated by the arrow):

SRT Home Page - Select MotoGP18
3

Start Recording Session

Click on the "Record new Session" button:

Record New Session Button
4

Begin Recording

Click on the Start button:

Start Recording Button
5

Race and Monitor

Go back to MotoGP and start your race. While racing, you will see SRT recording your lap data:

Live Recording in Progress
Important: Once you are done with the race, click the stop button in SRT to end the recording session.

Exporting Race Data to CSV

1

Select Session

Click on the session you would like to export:

Session Selection
2

Access Share Options

Click on the share button on the top right of the screen:

Share Button Location
3

Export to CSV

Click on the export to CSV option and save the file to your disk:

CSV Export Option

Importing CSV Data to SuperLap

1

Navigate to MotoGP Tab

Launch SuperLap, login, and navigate to the MotoGP tab:

SuperLap MotoGP Tab
2

Upload CSV File

Click on the upload CSV option and select the CSV file you saved:

CSV File Upload
3

Select Lap and Process

Select the lap number you would like to analyze, then click the Process button:

Lap Selection and Processing
4

Analyze Results

View the comparison between your racing line and the optimal line:

Racing Line Comparison Analysis

Analysis Features:

  • Line Thickness Sliders: Adjust thickness of borders, player line, and optimal racing line
  • Zoom Function: Use mouse scroll wheel to zoom in on specific track sections
  • Pan Navigation: Drag with mouse to move around the track view
