How to Login/Register

1

Launch SuperLap

When you launch SuperLap, you are greeted with the start page:

SuperLap Start Page
2

Access Login

If you wish to login, click on the login page. You can login with your existing username and password, or if you don't have an account, you can click the register button:

Login Page Interface
3

Register New Account

On the register page, enter your details as prompted and click register to complete registration:

Registration Form
Tip: After registration, you can go to the login page and login using the credentials you just set up.

Getting an Optimal Racing Line

1

Upload Track Image

On the dashboard tab you will see an upload image button. Click on it:

Dashboard with Upload Button
2

Select Track File

Click the upload image button and select a track image from your disk in the file menu that pops up:

File Upload Interface
3

Draw Centerline

After uploading, click on "trace centerline" to draw a rough centerline on the track.The point at which you start drawing and the direction you draw in, will be recorded as the start point and direction in the optimized race line

Drawing the Track Centerline
4

Process the Track

When you're happy with the centerline, click on the "process image" button:

Completed Centerline Ready for Processing
5

View Results

After processing finishes, you'll see the optimal racing line displayed on your track:

The red in the raceline shows the break points calculated by our system.

Generated Optimal Racing Line

Available Options:

  • Follow/Unfollow Cursor: Spacebar
  • Pan around the track: Click and Drag
  • Zoom in and out: Scrollwheel
  • Show more controlls: Hold Tab
Advanced Controls

Recording Laps in MotoGP

1

Launch Game

Launch MotoGP18, navigate to any race you wish to do and enter the race.

2

Record from SuperLap

Race and Monitor

Live Recording in Progress
Important: Once you are done with the race, click the Recording Game button to end the recording session.
